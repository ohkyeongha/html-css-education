글꼴 테스트

  • 글꼴
  • 폰트크기

font-family속성을 이용하여 글꼴을 지정한다.
작성법 -> font-family : '글꼴', 그다음글꼴

font-size 속성은 폰트의 크기를 지정하는 속성이다.
font-size : 16px / font-size : 1.5em;

text-shadow 속성은 글자의 그림자를 적용하는 속성
작성법 -> text-shadow : 1px 1px 1px #222; (가로, 세로, 번짐, 색상)

text-align 속성은 글자의 가로 정렬을 적용하는 속성
작성법 -> text-align: left | center | right | justify;

line-height 속성은 글자의 줄간격을 맞춰주는 속성
작성법 -> line-height : 2 | 10px; (배수단위, 픽셀단위)
